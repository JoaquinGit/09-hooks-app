# Guía Hooks

Material para repaso de Hooks:

- useState
- useEffect
- custom Hooks
- useRef
- useLayoutEffect
- useMemo
- useCallback


## Instalación 🔧

1. Clonar repositorio
    ```bash
    git clone
    ```
2. Instalar el proyecto
    ```bash
    npm install
    ```

## Instrucciones de uso

1. En src/index.js: cambiar nombre de componente a renderizar y descomentar su respectiva importación.

2. Correr
    ```bash
    npm start
    ```
